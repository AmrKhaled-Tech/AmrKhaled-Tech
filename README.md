### üí° Project Overview
This project demonstrates advanced Android development capabilities, moving beyond simple UI to handle background services and system-level broadcasts. The app provides users with real-time tools to manage their device performance and connectivity.

Instead of standard libraries, this app utilizes custom implementations for maximum efficiency:

* **Native Java Architecture:** Built entirely in Java for robust performance and memory management.
* **Background Services:** Implemented persistent `Services` and `BroadcastReceivers` to keep tools running even when the app is minimized.
* **Custom UI Components:** specific `RecyclerView` adapters with optimized layouts for smooth scrolling.
* **AdMob Integration:** Smart implementation of Banner and Interstitial ads (AdMob) ensuring high fill rates without disrupting user experience (UX).
* **Network Logic:** Advanced handling of connectivity states and data usage monitoring.
